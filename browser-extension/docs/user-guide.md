## 安裝指南

### Chrome 瀏覽器安裝

#### 方法 1：從應用商店安裝（推薦）
1. 前往 [Chrome Web Store](https://chrome.google.com/webstore)
2. 搜尋 "YouTube Video Manager"
3. 點擊 **"加到 Chrome"** 按鈕
4. 在彈出視窗中確認 **"新增擴充功能"**
5. 安裝完成後，Chrome 會自動啟用此擴充功能

#### 方法 2：手動載入（開發者模式）
1. 從 GitHub 下載專案程式碼
2. 在 Chrome 中輸入網址：`chrome://extensions/`
3. 啟用右上角的 **"開發者模式"**
4. 點擊 **"載入未封裝的擴充功能"**
5. 選擇專案中的 `browser-extension` 資料夾
6. 完成！擴充功能會在擴充功能列表中顯示

### Firefox 瀏覽器安裝

#### 方法 1：從 Add-ons 官方網站安裝（推薦）
1. 前往 [Firefox Add-ons](https://addons.mozilla.org)
2. 搜尋 "YouTube Video Manager"
3. 點擊 **"加入 Firefox"** 按鈕
4. 在彈出視窗中確認 **"新增"**
5. 安裝完成

#### 方法 2：手動載入（臨時）
1. 在 Firefox 中輸入網址：`about:debugging#/runtime/this-firefox`
2. 點擊 **"載入臨時附加元件"**
3. 導覽到專案中的 `manifest-firefox.json` 檔案
4. 選擇並確認
5. 完成（注意：重新啟動瀏覽器後會被移除）

### 驗證安裝成功

- 您應該在瀏覽器的工具欄看到 YouTube Video Manager 圖標
- 點擊圖標應該能看到擴充功能的 Popup 視窗

## 初次使用

### 步驟 1：開啟擴充功能

1. 前往任何 **YouTube 影片頁面**
2. 點擊瀏覽器工具欄上的 **YouTube Video Manager 圖標**
3. 會彈出一個小視窗（Popup）

### 步驟 2：登入 LINE 帳戶

Popup 會顯示登入介面：

```
┌─────────────────────────┐
│ YouTube Video Manager   │
│                         │
│   [LINE 登入]            │
│                         │
└─────────────────────────┘
```

**操作步驟：**

1. 點擊 **"LINE 登入"** 按鈕
2. 一個新的 LINE 登入視窗會彈出
3. 使用您的 LINE 帳號登入（如果尚未登入）
4. 允許應用程式授權（點擊 **"確認"** 或 **"允許"**）
5. 登入成功後，視窗會自動關閉，Popup 會顯示您的用戶信息

### 步驟 3：檢查登入狀態

登入成功後，Popup 會顯示：

```
┌─────────────────────────┐
│ YouTube Video Manager   │
│                         │
│ [用戶頭像] 您的名字      │
│                         │
│ [加入播放庫]            │
│ [加入播放清單]          │
│                         │
│ [設定]  [登出]          │
└─────────────────────────┘
```

恭喜！您已成功登入。

## 基本操作

### 操作 1：將影片加入播放庫

**適用情況：** 想要將 YouTube 影片保存到您的個人影片庫

**操作步驟：**

1. 在 **YouTube 影片頁面** 打開 Popup
2. 確認您已登入（看到用戶信息）
3. 點擊 **"加入播放庫"** 按鈕
4. 按鈕會變灰色並顯示 **"加入中..."**
5. 等待 1-3 秒

**成功訊息：**
```
✅ 已加入播放庫
```
訊息會在 3 秒後自動消失。

**可能的結果：**

| 訊息 | 含義 | 下一步 |
|------|------|--------|
| ✅ 已加入播放庫 | 成功 | 完成，影片已保存 |
| ⓘ 此影片已在播放庫中 | 重複 | 該影片已在庫中 |
| ❌ 網路連線失敗 | 網路問題 | 檢查網路並重試 |
| ❌ 請先登入 | 未登入 | 重新登入 |

### 操作 2：將影片加入播放清單

**適用情況：** 想要將 YouTube 影片加入特定播放清單

**操作步驟：**

1. 在 **YouTube 影片頁面** 打開 Popup
2. 確認您已登入
3. 點擊 **"加入播放清單"** 按鈕

根據您的設定方式，會出現不同的結果：

#### 情況 A：預設模式
- 影片會自動加入您在設定中選擇的播放清單
- 顯示訊息：**"✅ 已加入 [播放清單名稱]"**
- 無需手動選擇

#### 情況 B：自訂模式
1. 會彈出一個 **"選擇播放清單"** 對話框
2. 您會看到所有可用的播放清單列表
3. 每個播放清單會顯示：
   - 播放清單名稱
   - 目前包含的影片數量

**選擇播放清單：**

```
┌──────────────────────────────┐
│ 選擇播放清單                  │
│ ────────────────────────────  │
│ [✓] 我的最愛                  │
│     15 個影片                 │
│                              │
│ [ ] 稍後觀看                  │
│     8 個影片                  │
│                              │
│ [ ] 工作分類                  │
│     23 個影片                 │
│                              │
│               [取消]          │
└──────────────────────────────┘
```

4. 點擊要加入的播放清單
5. 等待確認訊息（1-3 秒）
6. 顯示 **"✅ 已加入 [播放清單名稱]"**

## 設定指南

### 開啟設定頁面

1. 點擊 YouTube Video Manager 圖標打開 Popup
2. 點擊右下角的 **"設定"** 按鈕
3. 設定頁面會在新標籤頁中開啟

### 設定頁面介面

```
┌────────────────────────────────┐
│ 設定                           │
│ YouTube Video Manager          │
├────────────────────────────────┤
│                                │
│ 播放清單模式                   │
│ ○ 預設模式                     │
│   自動加入到指定的預設播放清單  │
│ ◉ 自訂模式 (已選擇)            │
│   每次選擇要加入的播放清單      │
│                                │
│ [預設播放清單] (只在預設模式顯示) │
│ ▼ -- 選擇播放清單 --           │
│                                │
│ 播放庫設定                      │
│ ☐ 自動加入播放庫               │
│   加入播放清單時同時加入播放庫  │
│                                │
│ [儲存設定]  [重置為預設]       │
│                                │
└────────────────────────────────┘
```

### 設定步驟

#### 設定 1：選擇播放清單模式

**預設模式：**
- 適合有固定主要播放清單的使用者
- 操作更快，只需一鍵確認
- 需要預先選擇播放清單

**步驟：**
1. 選擇 **"○ 預設模式"** 單選按鈕
2. 下方會出現 **"預設播放清單"** 下拉菜單
3. 點擊下拉菜單選擇播放清單
4. 點擊 **"儲存設定"**

**自訂模式（推薦新使用者）：**
- 每次手動選擇播放清單
- 更靈活，可加入到不同播放清單
- 預設選項

**步驟：**
1. 選擇 **"◉ 自訂模式"** 單選按鈕
2. **"預設播放清單"** 部分會隱藏
3. 點擊 **"儲存設定"**

#### 設定 2：設定預設播放清單（僅限預設模式）

**步驟：**
1. 確保已選擇 **"預設模式"**
2. 點擊 **"選擇播放清單"** 下拉菜單
3. 從列表中選擇您最常使用的播放清單
4. 點擊 **"儲存設定"**

**提示：** 如果清單為空，表示您還沒有建立播放清單。請先在應用程式後台建立。

#### 設定 3：自動加入播放庫選項

**此選項的作用：**
- 啟用時：加入播放清單的同時會自動加入播放庫
- 停用時：只加入選定的播放清單

**步驟：**
1. 勾選 **"☑ 自動加入播放庫"** 複選框
2. 點擊 **"儲存設定"**

### 儲存設定

完成所有設定後：

1. 點擊 **"儲存設定"** 按鈕
2. 按鈕會顯示 **"儲存中..."**
3. 1-2 秒後會顯示 **"✅ 設定已儲存"** 確認訊息
4. 設定自動保存，重新啟動瀏覽器後仍然有效

### 重置設定

如果想恢復預設設定：

1. 點擊 **"重置為預設"** 按鈕
2. 會出現確認對話框：**"確定要重置為預設設定嗎？"**
3. 點擊 **"確定"** 確認
4. 所有設定會恢復為初始狀態

## 兩種模式詳解

### 預設模式

#### 什麼是預設模式？
自動將影片加入您預先設定的播放清單，無需每次選擇。

#### 何時使用？
- 您有一個主要播放清單用於保存大多數影片
- 您希望加快工作流程
- 您經常添加同一類型的影片

#### 使用流程

```
1. 在設定中選擇「預設模式」並指定播放清單
   ↓
2. 在 YouTube 影片頁面點擊「加入播放清單」
   ↓
3. 影片自動加入您的預設播放清單
   ↓
4. 顯示確認訊息：✅ 已加入 [播放清單名稱]
```

#### 優點
- ⚡ 速度快，一鍵確認
- 📖 簡單易用，無需思考
- 🎯 適合習慣性操作

#### 缺點
- 🔄 如果想加入不同播放清單，需進設定改變
- 📋 只能自動加入一個播放清單

---

### 自訂模式

#### 什麼是自訂模式？
每次添加影片時，在彈出的對話框中手動選擇播放清單。

#### 何時使用？
- 您有多個播放清單
- 不同類型的影片要加入不同清單
- 您希望有完全的靈活性

#### 使用流程

```
1. 在設定中選擇「自訂模式」
   ↓
2. 在 YouTube 影片頁面點擊「加入播放清單」
   ↓
3. 彈出「選擇播放清單」對話框
   ↓
4. 從列表中點擊目標播放清單
   ↓
5. 顯示確認訊息：✅ 已加入 [播放清單名稱]
```

#### 對話框說明

```
選擇播放清單
────────────────────────

[✓] 我的最愛（15 個影片）
  ↑ 點擊選擇此清單

[ ] 稍後觀看（8 個影片）
  ↑ 點擊選擇此清單

[ ] 工作分類（23 個影片）
  ↑ 點擊選擇此清單

────────────────────────
         [取消]
```

#### 優點
- 🔀 完全靈活，可加入任何清單
- 📚 適合多清單管理
- 🎨 適合組織不同類型的內容

#### 缺點
- ⏱️ 每次都需選擇，稍慢一點
- 🤔 需要記住清單名稱

#### 如何快速選擇？
- 使用鍵盤：按 **↑ ↓** 鍵移動，**Enter** 鍵確認
- 使用滑鼠：直接點擊目標播放清單
- 使用搜尋：（未來版本會加入）

## 訊息說明

### 成功訊息

#### ✅ 已加入播放庫
**含義：** 影片已成功加入您的影片庫
**操作：** 無需操作，訊息會自動消失

#### ✅ 已加入 [播放清單名稱]
**含義：** 影片已成功加入指定的播放清單
**操作：** 無需操作，訊息會自動消失
**例子：**
- ✅ 已加入 我的最愛
- ✅ 已加入 稍後觀看

#### ✅ 已加入 [清單名稱]（部分資訊無法取得）
**含義：** 影片已加入，但某些資訊（如高清縮圖）無法取得
**原因：** YouTube API 配額已用盡
**操作：** 無需操作，影片仍然已保存，只是縮圖可能不清晰

---

### 資訊訊息

#### ⓘ 此影片已在播放庫中
**含義：** 您嘗試加入的影片已存在於播放庫
**操作：** 可以選擇加入不同的播放清單
**建議：** 檢查是否已有此影片，避免重複

#### ⓘ 此影片已在播放清單中
**含義：** 此影片已存在於您選擇的播放清單
**操作：** 可以選擇加入其他播放清單
**建議：** 使用自訂模式加入不同清單

#### ⓘ 自訂播放清單模式將在下一版本推出
**含義：** 此功能還未完全實裝（舊版本）
**操作：** 更新至最新版本
**提示：** 最新版本已支援此功能

---

### 錯誤訊息

#### ❌ 無法取得影片資訊
**含義：** 無法從 YouTube 獲取影片資訊
**可能原因：**
- 影片已被刪除或設為私密
- 影片有年齡限制
- YouTube API 臨時故障

**解決方案：**
1. 檢查影片是否仍可在 YouTube 上觀看
2. 數分鐘後重試
3. 如果仍然失敗，請聯繫支援

#### ❌ 加入播放庫失敗
**含義：** 由於未知原因無法加入
**可能原因：**
- 網路連線不穩定
- 伺服器暫時故障
- 帳戶權限問題

**解決方案：**
1. 檢查網路連線
2. 等待幾秒鐘後重試
3. 確認您已登入
4. 如果持續失敗，請聯繫支援

#### ❌ 加入播放清單失敗
**含義：** 無法將影片加入所選播放清單
**可能原因：** 同上
**解決方案：** 同上

#### ❌ 網路連線失敗，請稍後再試
**含義：** 網路連線出現問題
**可能原因：**
- 網際網路連線中斷
- 伺服器無法連接
- 防火牆阻止連線

**解決方案：**
1. 檢查網際網路連線
2. 稍等幾秒後重試
3. 嘗試重新連接 WiFi 或行動網路
4. 檢查防火牆設定

#### ❌ 請先登入
**含義：** 您已登出或登入已過期
**可能原因：**
- Token 過期（24 小時後自動過期）
- 手動登出
- 瀏覽器清除了儲存資料

**解決方案：**
1. 關閉 Popup
2. 再次點擊 YouTube Video Manager 圖標
3. 點擊 "LINE 登入" 重新登入
4. 完成 LINE 授權

#### ❌ 無法開啟設定頁面
**含義：** 設定頁面無法載入
**可能原因：**
- 擴充功能檔案損毀
- 瀏覽器出現問題

**解決方案：**
1. 重新啟動瀏覽器
2. 如果仍未解決，卸載後重新安裝擴充功能

## 故障排除

### 常見問題與解決方案

#### 問題 1：看不到擴充功能圖標

**檢查清單：**
1. ✅ 確認已安裝擴充功能
2. ✅ 檢查瀏覽器工具欄右上角
3. ✅ 可能被隱藏在 **"⋯"** 菜單中

**解決方案：**
- Chrome：點擊工具欄的 **"拼圖"** 圖標，找到 YouTube Video Manager 並點擊圖釘固定
- Firefox：點擊右上角菜單，選擇 **"新增到工具欄"**

#### 問題 2：點擊圖標沒有反應

**檢查清單：**
1. ✅ 確認在 YouTube 頁面上
2. ✅ 等待 1-2 秒
3. ✅ 檢查網路連線

**解決方案：**
1. 重新整理網頁（F5 或 Ctrl+R）
2. 關閉並重新開啟 Popup
3. 重新啟動瀏覽器

#### 問題 3：無法登入

**檢查清單：**
1. ✅ 確認有有效的 LINE 帳號
2. ✅ 檢查網路連線
3. ✅ 確認 LINE 應用程式未被登出

**解決方案：**
1. 檢查網路連線
2. 在 LINE 應用程式中登出再登入
3. 清除瀏覽器快取（不會清除擴充功能資料）
4. 在無痕視窗中嘗試登入

#### 問題 4：加入影片後沒有反應

**檢查清單：**
1. ✅ 確認按鈕顯示 **"加入中..."**
2. ✅ 等待 2-3 秒
3. ✅ 檢查網路速度

**解決方案：**
1. 等待更長時間（網路慢時可能需要 5 秒）
2. 檢查瀏覽器主控台是否有錯誤（F12 > Console）
3. 重試操作

#### 問題 5：設定無法保存

**檢查清單：**
1. ✅ 確認點擊 **"儲存設定"** 按鈕
2. ✅ 等待 1-2 秒顯示確認訊息
3. ✅ 檢查網路連線

**解決方案：**
1. 重新點擊 **"儲存設定"**
2. 刷新頁面後重試
3. 清除瀏覽器快取
4. 重新啟動瀏覽器

#### 問題 6：影片加入後找不到

**檢查清單：**
1. ✅ 確認看到 **"✅ 已加入"** 訊息
2. ✅ 確認加入的是正確的清單
3. ✅ 檢查應用程式後台

**解決方案：**
1. 在應用程式中重新整理播放清單列表
2. 確認使用了正確的帳號
3. 檢查播放清單的排序或篩選設定

#### 問題 7：收到「網路連線失敗」

**檢查清單：**
1. ✅ 檢查網際網路連線
2. ✅ 檢查 WiFi 或行動資料
3. ✅ 檢查是否有防火牆阻止

**解決方案：**
1. 重新連接 WiFi
2. 切換行動資料開/關

## 常見問題

### 關於帳號與安全

**Q: 我的 Line 帳號密碼會被保存嗎？**
A: 不會。擴充功能只會保存授權後的 Token，不會保存密碼。

**Q: 我的資料安全嗎？**
A: 是的。Token 使用 AES-GCM 加密儲存，YouTube 影片資訊儲存在後端伺服器。

**Q: 我可以在多台裝置上使用嗎？**
A: 可以，但每台裝置需要分別登入。帳號資料儲存在後端，不同裝置可以同步。

**Q: 如何登出？**
A: 在 Popup 中點擊 **"登出"** 按鈕。本機儲存的 Token 會被刪除。

---

### 關於影片與播放清單

**Q: 我可以同時加入播放庫和播放清單嗎？**
A: 可以。先加入播放清單，再點擊「加入播放庫」。或者啟用「自動加入播放庫」設定。

**Q: 如果同一影片加入多次會怎樣？**
A: 擴充功能會自動偵測並顯示 **"此影片已在播放清單中"** 訊息，不會重複加入。

**Q: 我可以從擴充功能建立新播放清單嗎？**
A: 目前不支援，但已規劃在未來版本加入此功能。

**Q: 加入的影片會佔用我的空間嗎？**
A: 不會。影片儲存在後端伺服器，不會佔用您的裝置空間。

---

### 關於功能與支援

**Q: 為什麼某些 YouTube 影片無法加入？**
A: 可能是因為：
- 影片已被刪除
- 影片設為私密或受限
- 影片有年齡限制
- 您無此影片的存取權限

**Q: 擴充功能支援哪些語言？**
A: 目前支援繁體中文和英文。更多語言將在未來版本推出。

**Q: 擴充功能會減慢我的瀏覽速度嗎？**
A: 不會。擴充功能使用最小化的資源，只在您點擊時啟動。

**Q: 我可以停用擴充功能嗎？**
A: 可以。在擴充功能管理頁面將其停用。重新啟用時不會丟失資料。

**Q: 如何卸載擴充功能？**
A: 在擴充功能管理頁面點擊擴充功能上的 **"移除"** 按鈕。

---

### 關於未來功能

**Q: 未來會加入什麼功能？**
A: 計劃中的功能包括：
- 批次操作（同時加入多個影片）
- 鍵盤快捷鍵
- 進階搜尋和篩選
- 影片推薦
- 深色模式

**Q: 我可以建議新功能嗎？**
A: 可以！請在 GitHub Issues 中提交功能建議。

**Q: 什麼時候會推出新版本？**
A: 通常每月推出一次更新。重大功能可能需要更長時間開發。

## 安全建議

### 密碼與登入安全

✅ **務必：**
- 在官方應用程式商店安裝擴充功能
- 定期更新到最新版本
- 使用強密碼和雙重認證保護 LINE 帳號
- 在公共電腦上登出

❌ **避免：**
- 在公共 WiFi 上執行敏感操作
- 分享您的授權連結或 Token
- 在同意隱私時不閱讀條款
- 在已被入侵的電腦上使用
